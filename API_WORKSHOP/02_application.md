# Application

## Definition

An **Application** can be considered as a **set of codes and instructions** that performs a **specific task**.

An application works by **taking input, processing the data, and producing an output**.

```
Input → Processing → Output
```

Example:

- Calculator Application
- Banking Application
- Student Management System
- Web Browser

---

## Types of Applications

Applications are mainly divided into **two types**:

1. **Standalone Application**
2. **Client-Server Application**

---

## 1. Standalone Application

A **Standalone Application** is a software program that **runs on a single computer system** and **does not require a network connection** to operate.

These applications are installed directly on the **user's local machine** and all processing happens **within that system**.

### Characteristics

- Runs on a **single system**
- **No internet required**
- Processing happens **locally**
- Data is stored **locally**

### Examples

- Calculator
- Notepad
- MS Word
- VLC Media Player

---

## 2. Client-Server Application

A **Client-Server Application** is an application where **multiple clients communicate with a central server** through a **network**.

The **client sends a request**, and the **server processes the request and returns a response**.

```
Client → Request → Server
Client ← Response ← Server
```

### Characteristics

- Requires **network communication**
- **Multiple users** can access the system
- Data is stored on the **server**
- Processing can happen on **server or client**

### Examples

- Banking systems
- Email systems
- Online shopping platforms
- Social media platforms

---

## Web Application

A **Web Application** is a type of **Client-Server Application** that runs inside a **web browser**.

Users access web applications through the **internet using browsers** like Chrome, Edge, or Firefox.

### Examples

- Gmail
- Facebook
- Amazon
- Online Banking Systems

---

## Summary

| Application Type | Description |
|------------------|-------------|
| Standalone Application | Runs on a single computer without network |
| Client-Server Application | Client communicates with server over a network |
| Web Application | A client-server application accessed through a web browser |
