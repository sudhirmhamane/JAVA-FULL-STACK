# Basics of Internet

The internet, sometimes simply called the net, is a world wide system of interconnected computer network and electronic devices that communicate with each other using an established set of protocols.

The internet was conceived by the "Advanced Reasearch Project Agency"(ARPA) of as government in 1969.

---

## Advanced Research Project Agency(ARPA):

ARPA was created after the "sputnik" launch by the sovient union.

sputnik was the first man made satelliate released in the space.

The Us government wanted to improve scientific and technological research.

It was created in 1958 by united state departement of defence.

It's main goal was to develop technology and research project for military purpose.

### ARPA contribution:

ARPA created the first packet-switching network called as 'ARPANET'.

This network later evolved into the modern internet.

#### Features of 'ARPANET':

1. Used packet switching
2. Connected universites and reasearch lab.
3. Allowed computers to communicate with each others.
