# Basics of Programming:

## Language:

A language is the mode or medium of communication between two or more entities(object).

It is used to share information

Language are classifed based on the types of entity.

## Programmer:

A person who has knowledge of programming language is know as programmer.

## Programming:

A Process of Creating set of instruction using programming language is know as programming.

## Programming Language:

A Language which is used to command a machine or computer to perform a specific task is know as programming language.

## Source Code:

A file created by programmer which contains set of instruction or block of code is know as source code.

every source code will have an language extension(filr name)

for ex: java-> .java, python-> .py, javascript-> .js, .mjs, .cjs, etc

## Binary Language:

A Language which is made up of only two charactors 0's and 1's is know as binary language.

This binary charactors are typically group in a set of 16 or 32.

Binary language is only understandable to m/c that way we also called binary as m/c understandable language.
