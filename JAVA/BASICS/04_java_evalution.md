## BETA Version:

Beta version is a stage in software application development where the application is almost completed but it is released to a limited group of user for the real world testing befor the final release.

## Purpose of BETA:

- To test the software in real world condtions
- To take the feedback from the users
- To identify and fix the bugs
- To improve performance and useability

Java releases it's beta version in 1995

## Versions of JAVA:

Sun Microsystem introduced first stable version of java in jan 1996 i.e JDK 1.0

Till date we have 26 different versions of java

every year new java version are introduced in the month of march and septemeber.
