# LTS and Editions in java

## Features of JAVA 8:

Java fist LTS version was introduced in year 2014 which was JDK 8

This version contains several new implementation such as,

1. Lambda Expression
2. Stream API
3. Function Enterprise
4. Date and Time API
5. Method Interface, etc

---

## Standard Edition:

Java se is used to develop desktop application, consol based app or backend logic

It provide fundamental API's Libraries and tools required to build a java program

## Enterprise Edition

Java EE (also know as Jakarta Enterprise Edition J2EE) is an extension of java's standard edition which is used to develop,

1. Enterprise Application
2. Large Scale Application

For Ex -> Banking system, e-commerce web, erp system, web appliation

## Micro Edition:

Java ME is a version desing for small devices with limited memory and processing power

we can develop different kind of application such as,

1. mobile application
2. embedded system
3. IOT devices
4. Set-up boxes, etc
