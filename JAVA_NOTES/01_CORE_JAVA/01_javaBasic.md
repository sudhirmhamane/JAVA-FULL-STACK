# JAVA

## Introduction

Java is a **high level, object oriented, platform independent** programming language.

Java was developed by **James Gosling** and his team at **Sun Microsystem** *(SUN stands for Stanford University Network)* in **1991**.

Currently Java is owned by the **Oracle Corporation**.

- Java's **First Beta version** was released in the year **1995**.
- Java's **First version** was introduced in **Jan 1996** and it was **JDK 1.0**.
- Java has **5 LTS (Long Term Support)** versions: **Java 8 / 11 / 17 / 21 / 25**

---

## Java Editions

Java has **3 editions** which are used to develop different kinds of applications:

| # | Edition | Full Name |
|---|---------|-----------|
| 1 | **SE** | Standard Edition |
| 2 | **EE** | Enterprise Edition |
| 3 | **ME** | Micro Edition |

## Features of Java

### 1. High Level Language
Java is easily understandable by humans because it is **readable, understandable and instructable** by all.

---

### 2. Platform Independent Language
Applications developed using Java on one platform can **run on multiple platforms** without recompilation and modification.

---

### 3. Statically Typed Language
Types in Java have to be **mentioned explicitly** before using them at the compilation time.

---

### 4. Dynamic Memory Allocation
Java has a **Garbage Collector** which removes unused or non-referenced objects from the heap area and frees up space.

---

### 5. Multi-Threaded Language
Java supports **multithreading** — we can create multiple threads and execute them simultaneously.

---

### 6. Object Oriented Language
In Java we can create **objects using classes**, reuse them, and manipulate them to get an expected output.

---

### 7. Portable Language
Java follows **"WORA — Write Once Run Anywhere"** which improves portability of using it across multiple platforms.

---

### 8. Robust
Java is a **robust language** — it provides a mechanism to handle unwanted events which occur at runtime.

It avoids system crashes in complex applications with the help of **exception handling**.

---

### 9. High Security
Java provides a **controlled and safe execution environment** through features like:

- No Pointer
- Byte Code Verification
- Automatic Memory Management
- Type Safety
- Access Modifiers
