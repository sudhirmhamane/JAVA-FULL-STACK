# JVM:

JVM stands for Java Virtual Machine,

It's the part of JRE which is used to run the java program.

The Byte code generated by compiler is executed by the jvm.

JVM Makes the JAVA as Platform independent language, one byte code can run on any or multiple platform which machine installed JVM.

The platform which is used to build and run java applications(JDK, JRE, JVM) are the Platform dependent.

For every different operating system we have different JRE/JDK.

### Task of JVM:

1. Accept Byte Code
2. Verifies the Byte Code
3. Class loading process
4. Execution Process
memory management
thread management
