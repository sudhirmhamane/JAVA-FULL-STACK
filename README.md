# JAVA-FULL-STACK 🚀

A structured **Java Full Stack Development repository** designed for learning and revision, starting from **SQL fundamentals** 
and gradually expanding frontend, backend and full-stack concepts.

This repository is built in a **topic-wise, notes + examples format**, making it beginner-friendly as well as useful for interview preparation.

---

## 📌 Repository Roadmap

This project follows a **step-by-step learning path**:

1. **SQL (Current Phase ✅)**
2. Frontend (HTML, CSS, JavaScript, React)
3. Java Core
4. Advanced Java
5. JDBC
6. Servlets & JSP
7. Spring & Spring Boot
8. REST API
9. Full Stack Integration
10. Projects & Assignments
