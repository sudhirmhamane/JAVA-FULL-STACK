# DBMS (Database Management System)

**DBMS** is software used to **maintain and manage the database**.

## DBMS provides major features
- **Security**
- **Authorization**
- **Backup**
- **Recovery**

### Example
We use a **query language** to communicate with DBMS software.

# Data vs Information

## Data
| Points |
|--------|
| Raw facts & figures |
| Does not have context |
| May be unprocessed |
| Used as input |
| Can be numbers, text, or symbols |
| Collected from various sources |
| Stored and unorganized |

## Information
| Points |
|--------|
| Processed data |
| Has meaningful context |
| Useful for decision making |
| Output of processing |
| Usually presented in readable form |
| Designed for data analysis |
| Organized and structured |
