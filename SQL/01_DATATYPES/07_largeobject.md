# large object datatype

It is a type of data which used to store large obj like.

1. character Large Object
- It is a type of data used to store character upto 4GB of Size.
- **Syntax** - CLOB

2. Binary Large Object
- which used to store binaries (0, 1) of any Image, Audio, Video, Doc upto 4 GB of Size.
- **Syntax** - BLOB
