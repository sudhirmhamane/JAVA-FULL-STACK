# Not Null constraint

- It is type of Constraint we assign to the particular column which will not to be null 
- while inserting the data, data must be inserted not remain empty.
- Null values are not allowed in this constraint.
- Used to ensure mandatory field.
- It applied only on individual column.
- Table can have multiple Not Null constraints.
- Not Null not automatically creates index.
- Ex- Name, sal, Dept name
