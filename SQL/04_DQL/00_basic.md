# Data Query Language (DQL)

It is the statement used to fetch the data from database or table.

their are 3 methods to fetch the data from the database or table.

## 1. Projection

- It is the process of feaching the data from table by selecting the only columns.

## 2. Selection

- It is the process of feaching the data from table by selecting the both rows and columns.

## 3. Joints

- It is the process of feaching the data from all the multiple tables simultaneously is know as joints.
