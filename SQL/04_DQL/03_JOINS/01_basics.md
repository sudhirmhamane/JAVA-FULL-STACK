# Joins in the sql:

It is the process of fetching the data from two or multiple tables simultenously is known as the Joins

The joins are classified into the 5 types,

1. Cartesian Join or Cross Join
2. Inner Join
3. Natural Join
4. Self Join
5. Outer Join:
    1. Left Outer Join
    2. Right Outer Join
    3. Full Outer Join

---

1. Whenever we joints the multiple tables, first all the tables will get combines together it will become only one table.

2. Based on the record which we are going to get in result table the joins has been classified into 5 types.

3. There are three types of the records which we get if we join the multiple tables.

- Error Records
- Matching Records
- Un Matching Records

---

1. Matching Records:

The primary key value from the parent table if that is present in the child table as foreign key this type of record is known as Matching Records.


2. Un Matching Records:

The primary key value from the parent table if that is not present in the child table as foreing key this type of record is knowm as Un Matching Records.


3. Error Records:

The each and every records from the one table if that are marging with the each and every records from the another table this types of records are known as Error Records.
