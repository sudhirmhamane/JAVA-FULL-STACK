# 3 Tier Architecuture

3 Tier Architecuture mainly consits of,

1. Frontend:-
- Is the user interface fo application.
- It is the part that users can see and interact with.
- Ex- login page, sing up page
- Technology used - HTML, CSS, JAVASCRIPT

2. Backend:-
- It is the sever side part of the application.
- It works in background.
- It process requests and send response.
- Technology used- Java, Python, Php, Node.js

3. Database:- 
- Database is place where data is stored permanently
- It stores user information, passwords, posts and transactions.
- Ex- In an online-banking app
        - your account balance
        - transaction history all are stored in a database.
- Technology used- Mysql, NoSql, MongoDB
