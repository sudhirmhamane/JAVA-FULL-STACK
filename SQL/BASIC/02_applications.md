# Applications

- The set of instructions which are use to perform and only specific task is kown as applications.

- The applications are also called as software systems or application.

- Applications are divided into three major types as follows,

1. Standalone Application:-

- The application which does not requires any internet connectivity to access it are know as standalone application.
- No internet is required.
- Database is optional.
- Multiple users can't able to access at a time.
- Installations are mandatory.
- Ex- Calculator, Calender, Clock, Reminder, Notes, Excel, Word, etc.

2. Web Applications:-

- The web application which we can access through on any search engine are know as web applications.
- The internet connection is mandatory.
- Database is also required.
- Multiple user can access at a time.
- Installation is not mandatory.
- Ex- www.facebook.com, www.google.com etc

3. Client Server Applications:-

- The application which we able to access through client server this types of application called as client server application.
- Internet, Database connectivity is required.
- Multiple users can access.
- Installation is mandatory.
- Ex- Banking application, Gaming applications etc.
