# Unique Constraint

- It it the type of Constraint we assign for particular column which will not allows to insert duplicate or reapeated values in the table.
- **Syntax** - UNIQUE
- one null values is allowed in the Unique constraint.
- used to maintain the uniqueness of data.
- It can be applied single or multiple column.
- A table can have multiple unique constraints.
- Automatically creates the a unique index.
- EX- email, adhar number
