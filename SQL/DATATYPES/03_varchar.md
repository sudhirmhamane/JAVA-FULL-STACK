# varchar data type

It is the type of data which is used to store characters like,

- Alphabates (a-z, A-Z)
- Numbers (0,1,2....,9)
- Special Char ($, #, @, %)
- **Syntax** - varchar(size)
- Ex- varchar(10)
    vikas(6), ram(3), raj(3)
- We can store max 2000 char by using varchar datatypes.
- There is no wastage of memory in varchar.
- It follows the concept of variable length memory allocation.
- Inital and final blocks may or may not be same or fixed.
